# Ubuntu

> Installing Dependencies

```
sudo apt install git cmake mingw-w64
```

> Installing Magic_Recode

```
git clone https://github.com/qwertyuiop3/Magic_Recode.git --recursive

cd Magic_Recode

./Build.sh
```

# Arch Linux

> Installing Dependencies
